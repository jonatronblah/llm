# LLM/RAG Template for Python

## Using
- Huggingface Transformers
- llamma-cpp-python
- llamaindex
- pgvector

## Requirements
- Docker + NVIDIA Container Toolkit
- WSL(2) or Linux

## Get Started
- create a .env in project root with the following environment variables:
    - POSTGRES_USER
    - POSTGRES_PASSWORD
    - POSTGRES_DB
